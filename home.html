Linguagem de Programação

A primeira disciplina do semestre foi Linguagem de Programação, na qual aprendemos, utilizando a IDE Visual Studio Code, conceitos como a linguagem de marcação HTML. Por exemplo, exploramos as tags mais usadas atualmente, estruturas adequadas, HTML semântico e tabelas, entre outros. Também abordamos a estilização com CSS, aprendendo conceitos de organização de elementos usando o flexbox. Após isso, iniciamos alguns projetos, como a criação de um currículo pessoal, para unir essas duas tecnologias.
para facilitar para pessoas que tiverem interesse nos projetos, a grande maioria foi enviado para o google drive, portanto basta acessar o link! Arquivos do projeto currículo: click aqui!
Ao longo das aulas, os professores trouxeram alguns exercícios de lógica de programação utilizando a linguagem JavaScript, onde foi possível aprender mais sobre estruturas comuns, como "if", "for", "while", "switch case", entre outros. Esses primeiros exercícios tinham como foco principal o uso de operações matemáticas. Alguns exemplos de exercícios em JavaScript: click aqui!
Depois de vários conceitos já abordados em aula, foi requisitado um projeto com o tema "folha de pagamento" para implementação das três tecnologias: HTML, CSS e JavaScript. Esse projeto foi realizado em squads, tentando simular a metodologia Scrum, que já havia sido abordada nas primeiras aulas. Veja o projeto: click aqui!
Ao final da disciplina, houve um desafio bastante interessante, no qual consistia em criar um site que aplicasse uma estilização pronta a um arquivo HTML escolhido pelo usuário. O maior desafio foi criar um script que utilizasse a árvore DOM e fosse funcional. Os arquivos desse desafio: click aqui!

Automação Industrial

O segundo tema abordado no curso foi Automação Industrial, no qual foi possível conhecer um pouco mais sobre módulo embarcado, sensores, CLP e robótica, que são comumente utilizados em ambientes industriais. Aprendemos um pouco da sintaxe da linguagem C++ utilizando a plataforma Tinkercad da Autodesk, onde é possível realizar diversos tipos de simulação. Nessa plataforma, conseguimos simular o uso do Arduino e programar vários projetos pequenos para entender os princípios de um microcontrolador. Os alunos tiveram a oportunidade de conhecer um pouco mais sobre a robótica e como ela funciona geralmente nesse ambiente, com braços e eixos, além da própria programação. Por fim, no laboratório, foi possível manipular um braço robótico na prática.
